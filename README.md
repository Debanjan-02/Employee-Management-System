# Employee Management System (Java + JDBC + MySQL)

A console-based CRUD application built using Core Java, JDBC, and MySQL. 
This project demonstrates how to interact with a relational database 
using JDBC APIs and is structured following clean code and SDLC principles.

## Features
- Add new employee
- View all employees
- Update employee details (name, salary, department)
- Delete employee records
- Input validation & exception handling
- Secure SQL operations using PreparedStatement

## Tech Stack
- Java
- JDBC
- MySQL
- DAO Design Pattern

## How to Run
1. Import project into IntelliJ/Eclipse.
2. Create MySQL table using database.sql.
3. Update DBConnection.java with your DB credentials.
4. Run Main.java
